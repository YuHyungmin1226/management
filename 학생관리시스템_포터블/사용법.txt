🎓 학생 관리 시스템 - 간단 사용법

🚀 시작하기
1. 학생관리시스템.exe 더블클릭
2. 브라우저에서 http://localhost:5000 접속

📝 기본 사용법
• 학생 추가: 상단 메뉴 → "학생 추가"
• 평가 추가: 학생 상세보기 → "새 평가 추가"
• CSV 업로드: 상단 메뉴 → "CSV 일괄 등록"
• 데이터 다운로드: 상단 메뉴 → "평가 다운로드"

🔤 한글 자모 지원
• 자모 조합 입력 완벽 지원 (ㄱ + ㅏ = 가)
• 영문 혼용 이름 가능 (김John)
• 실시간 입력 검증

🎨 깔끔한 인터페이스
• 과도한 시각적 효과 제거
• 일관된 버튼 스타일
• 직관적인 레이아웃

⚠️ 주의사항
• Windows Defender에서 실행 허용
• 방화벽에서 포트 5000 허용
• 데이터 백업 권장

📞 문제 발생 시
• GitHub 이슈 등록
• 로그 파일 확인 (management.log)
