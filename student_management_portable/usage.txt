��📚 학생 관리 시스템 - 포터블 버전

🚀 빠른 시작

## macOS
1. 터미널에서: ./student_management_mac
2. 브라우저에서: http://localhost:5003

## Windows
1. 더블클릭: student_management.exe
2. 브라우저에서: http://localhost:5003

📝 주요 기능
• 학생 추가/수정/삭제
• 평가 관리 (과목별 점수)
• CSV 일괄 등록/다운로드
• 실시간 검색
• 한글 자모 완전 지원

⚠️ 주의사항

## macOS
• 첫 실행 시 "확인되지 않은 개발자" 경고 → 시스템 환경설정에서 "허용"
• 방화벽에서 포트 5003 허용

## Windows
• Windows Defender에서 실행 허용
• 방화벽에서 포트 5003 허용

🚨 포트 충돌 시
다른 포트로 실행:
• macOS: FLASK_RUN_PORT=5004 ./student_management_mac
• Windows: FLASK_RUN_PORT=5004 student_management.exe
• 접속: http://localhost:5004

📞 문제 발생 시
• 로그 파일 확인 (management.log)
• 터미널/명령 프롬프트에서 실행하여 오류 메시지 확인
